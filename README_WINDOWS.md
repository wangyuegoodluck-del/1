# Windows 同事使用说明

## 第一次使用

1. 解压压缩包。
2. 先安装 Node.js LTS：<https://nodejs.org/>
3. 双击 `install_windows.bat`。
4. 安装完成后，双击 `start_windows.bat`。
5. 浏览器会打开：

```text
http://127.0.0.1:3000/
```

## 以后每次使用

直接双击：

```text
start_windows.bat
```

启动后不要关闭黑色命令窗口；关闭窗口后系统也会停止运行。
