# 法奥机器人智能合同系统 - 同事使用说明

## 运行方式

1. 安装 Node.js。
2. 在本文件所在目录打开终端。
3. 第一次运行先执行：

```bash
npm install
```

4. 启动系统：

```bash
npm run dev
```

5. 浏览器打开：

```text
http://localhost:3000/
```
